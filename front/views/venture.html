ABOUT

VentureBound is a full stack application that allows a user to create an account, securely log in, create adventures and save them along with images, Google Maps coordinates and other info. A user can view all saved adventures both as markers on a Google Maps instance, and as adventure titles.

Tech

  • HTML/CSS
  • JavaScript
  • Angular
  • Angular UI Router
  • TweenMax
  • Google Maps API
  • Node
  • PostgreSQL
