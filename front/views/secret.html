ABOUT

wearetopsecret.cf is a clone of wearetopsecret.com. A couple other devs and I were wowed by the graphics and effects on Super Top Secret's website. So we set out to see how much of it we could recreate in a short amount of time. We wrote custom CSS animations to do things that we hadn't thought were possible with CSS. We wrote our own JavaScript parallax functions, and we created TweenMax animations. And we had a super fun time doing it.

Tech

  • JavaScript
  • Angular
  • Angular UI Router
  • TweenMax
  • SASS
  • Node
